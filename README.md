# Data Analyzer
Data analyzer is a proof of concept program to show how AI can fit into analyst workflows. The prototype takes the data you provide as an `xlsx`, gives you 3 critical takeaways and makes graphs to support those takeaways.
The prototype is best used as a tool to vector your attention and give you ideas to begin your analysis.

## Up front disclaimers
- This program will send your data to genai.mil -> **APPROVED FOR IL5 - CONTROLLED UNCLASSIFIED INFORMATION (CUI) - NO PII/PHI**.
- AI responses can and will be incorrect in many cases. Never use anything generated by AI directly without thoroughly verifying it is correct.
- It is **VERY STRONGLY** recommended that you run the programs using `run/run.bat` and `run/run previous run.bat`. AI generated code could theoretically spill CUI data by doing things such as publishing it to github automatically through git CLI. The scripts run the program using Docker, which containerizes the execution of code and should prevent any possible spillage.

## Setup
1. On Windows, download the repo then run `setup/setup.bat`. This will install Python, the required Python packages and configure the environment.
2. Delete or move `data/sample_data.xlsx` and place your data in the `data` folder. (Or play around with the sample data).

## Running the program *DO IT THIS WAY TO GUARANTEE CUI CAN'T SPILL*

- `run.bat` will generate new analysis from genai.mil.
- `run previous run.bat` will run the code generated by the previous run if you want to look at it again.

## Uninstalling things installed by setup

1. Run `setup/uninstall.bat`
